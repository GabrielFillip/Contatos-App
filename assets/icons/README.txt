ÍCONES NECESSÁRIOS PARA O APP DE CONTATOS
==========================================

Adicione os seguintes ícones nesta pasta:

1. person_placeholder.png (192x192px)
   - Usado como avatar padrão quando não há foto
   
2. empty_contacts.png (256x256px)
   - Ilustração para estado vazio da lista
   
3. app_logo.png (512x512px)
   - Logo do aplicativo

Formatos aceitos: PNG com fundo transparente
Resolução: @2x e @3x para melhor qualidade

Após adicionar os ícones, descomente as linhas no pubspec.yaml:
# assets:
#   - assets/icons/
